# Iris Flower Classification

## Objective
To classify Iris flowers into different species using Machine Learning techniques.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-Learn
- VS Code

## Dataset
Iris.csv

## Algorithm Used
Random Forest Classifier

## Steps Performed
1. Loaded the dataset
2. Preprocessed the data
3. Trained the machine learning model
4. Predicted flower species
5. Evaluated model accuracy
6. Visualized data using graphs

## Outcome
Successfully classified Iris flowers with high accuracy.
