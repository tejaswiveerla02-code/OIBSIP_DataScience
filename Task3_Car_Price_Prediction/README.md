# Car Price Prediction Using Machine Learning

## Objective
To predict the selling price of cars using machine learning techniques based on various vehicle attributes.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-Learn
- VS Code

## Dataset
car data.csv

## Algorithm Used
Random Forest Regressor

## Features Used
- Year
- Present Price
- Driven Kilometers
- Fuel Type
- Selling Type
- Transmission
- Owner

## Steps Performed
1. Loaded the dataset
2. Performed data preprocessing
3. Encoded categorical variables
4. Visualized feature correlations
5. Split data into training and testing sets
6. Trained a Random Forest Regressor model
7. Predicted car prices
8. Evaluated model performance using MAE, MSE, and R² Score

## Results
- MAE: 0.614
- MSE: 0.839
- R² Score: 0.964

## Outcome
Successfully developed a machine learning model capable of predicting car prices with high accuracy.
